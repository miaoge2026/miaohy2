## 🔧 快捷命令

安装完成后，会自动创建`hy2`快捷命令：

```bash
hy2 start     # 启动服务
hy2 stop      # 停止服务
hy2 restart   # 重启服务
hy2 status    # 查看状态
hy2 logs      # 查看日志
hy2 config    # 编辑配置
hy2 client    # 查看客户端配置
hy2 qr        # 生成二维码
hy2 test      # 测试配置
hy2 uninstall # 卸载服务
```
