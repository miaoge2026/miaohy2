# Hy2 - Hysteria 2 Installation Script

🚀 **One-click installation script for Hysteria 2**

## 🎯 Quick Start

### Fusion Version (Recommended)
```bash
bash <(curl -sL https://raw.githubusercontent.com/miaoge2026/miaohy2/main/scripts/hy2-fusion.sh)
```

### Custom Installation
```bash
bash <(curl -sL https://raw.githubusercontent.com/miaoge2026/miaohy2/main/scripts/hy2-fusion.sh) --domain your-domain.com
```

## ✨ Features

- 🎯 **One-click installation** - Zero configuration required
- ⚡ **30-second deployment** - Lightning fast installation
- 🔧 **Smart optimization** - Automatic system tuning
- 📱 **Quick commands** - hy2 command for easy management
- 📊 **Subscription conversion** - Multiple format support
- 🔒 **Secure defaults** - Production-ready security

## 🔧 Quick Commands

After installation:
```bash
hy2 start     # Start service
hy2 stop      # Stop service
hy2 restart   # Restart service
hy2 status    # Check status
hy2 logs      # View logs
hy2 uninstall # Uninstall service
```
