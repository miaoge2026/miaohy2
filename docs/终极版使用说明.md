# 终极版Hysteria安装脚本使用说明

## 🎯 概述

**终极版**是喵哥Hysteria安装脚本的最简化版本，专为"一键安装"设计。它去掉了复杂的菜单和交互，保留了核心功能，让安装过程变得极其简单。

## 🚀 快速开始

### 一句话安装
```bash
# 最简单的方式
bash <(curl -sL https://raw.githubusercontent.com/miaoge2026/miaohy2/main/scripts/喵哥一键安装_hysteria_终极版.sh) --auto
```

### 使用wget
```bash
wget -O hy.sh https://raw.githubusercontent.com/miaoge2026/miaohy2/main/scripts/喵哥一键安装_hysteria_终极版.sh
chmod +x hy.sh
./hy.sh --auto
```

## 📋 安装选项

### 自动模式
```bash
# 使用所有默认值
./hy.sh --auto

# 或者使用环境变量
AUTO=true ./hy.sh
```

### 自定义安装
```bash
# 自定义端口和密码
./hy.sh --port 443 --password mypass

# 自定义域名
./hy.sh --domain your-domain.com

# 完整自定义
./hy.sh --port 443 --password mypass --domain your-domain.com --up-bw 100 --down-bw 20
```

### 环境变量方式
```bash
# 使用环境变量配置
PORT=443 PASSWORD=my_strong_password DOMAIN=your-domain.com ./hy.sh

# 自动模式
AUTO=true ./hy.sh
```

## 🔄 卸载

### 使用脚本卸载
```bash
./hy.sh --uninstall
```

### 手动卸载
```bash
# 停止服务
systemctl stop hysteria
systemctl disable hysteria

# 删除文件
rm -f /usr/local/bin/hysteria
rm -f /etc/systemd/system/hysteria.service
rm -rf /root/hysteria

# 重新加载systemd
systemctl daemon-reload
```
